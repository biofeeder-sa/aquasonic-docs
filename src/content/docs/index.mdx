## ¿Qué es Aquasonic?

**Aquasonic** es un sistema de monitoreo y control desarrollado con Angular 17+ por el equipo
de **biofeeder-sa**. Integra dashboards en tiempo real, gestión de dispositivos y comunicación
con una API REST propia.
